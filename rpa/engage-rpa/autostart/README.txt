Autostart options:

1. user
   Use this if you do not have admin access.
   The script starts automatically when the Windows user logs in.

   Install:
   user\install_user_startup.bat

   Uninstall:
   user\uninstall_user_startup.bat

2. admin
   Use this if you have permission to create Task Scheduler tasks.
   The script starts automatically when the Windows user logs in.

   Install:
   admin\install_autostart.bat

   Uninstall:
   admin\uninstall_autostart.bat

Logs:
..\logs\scheduler.log
